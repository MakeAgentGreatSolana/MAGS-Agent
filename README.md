## About

**MAGS** is a community-driven Solana agent built for blockchain interactions. It provides a powerful toolkit for:
- 🤖 Automated blockchain operations
- 💰 SOL transfers and balance management
- 🔐 Secure wallet management
- 📊 Real-time blockchain monitoring

## Features

- ✅ **Wallet Management** - Generate, import, and manage Solana wallets
- ✅ **Transaction Handling** - Send SOL and interact with smart contracts
- ✅ **Real-time Monitoring** - Track blockchain state and wallet balance
- ✅ **CLI Scripts** - Ready-to-use scripts for common operations
- ✅ **TypeScript** - Full type safety and modern JavaScript features

## Quick Start

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env with your wallet private key
```

### 3. Run the agent

```bash
npm run dev
```

## Available Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Run agent in development mode |
| `npm start` | Run production build |
| `npm run build` | Compile TypeScript |
| `npm run check-balance` | Check wallet balance |
| `npm run transfer` | Transfer SOL |
| `npm run generate-wallet` | Generate new wallet |
