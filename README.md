## About

MAGS is a community-driven project built on the Solana blockchain, celebrating the intersection of technology, creativity, and decentralized innovation.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Solana CLI
- A Solana wallet (Phantom, Solflare, etc.)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/mags.git

# Navigate to project directory
cd mags

# Install dependencies
npm install
```
